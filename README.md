# ながのケーキ

長野県にある洋菓子店「ながのCAKE」の商品を通販するためのECサイト開発。<br>
SNSでバズり、全国から問い合わせが殺到したことが経緯。<br>

## 制作期間
2022/4/13~4/25

## 制作メンバー
- ちこ
- しげちゃん
- KT

		ログイン機能<br>
		ログアウト機能<br>
		商品一覧表示機能<br>
		商品詳細情報表示機能<br>

		カート一覧機能<br>
